## 📖 Overview

This repository contains the source code for my personal portfolio website.

The portfolio serves as a central hub where recruiters, hiring managers, and fellow developers can explore my technical background, software engineering experience, QA automation expertise, and real-world projects. The website is fully responsive, lightweight, and deployed automatically using GitHub Pages with GitHub Actions.

## ⚙️ Deployment

The portfolio is hosted using **GitHub Pages**.

Every push to the main branch automatically triggers a GitHub Actions workflow, which builds and deploys the latest version of the website.

This enables continuous deployment with minimal manual effort.
